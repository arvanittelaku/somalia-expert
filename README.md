# Somalia Expert Witness

SEO-optimized lead-generation website for [somaliaexpert.com](https://www.somaliaexpert.com).

## Stack

- Next.js 16 (App Router), TypeScript, Tailwind CSS v4
- Formspree contact form (`NEXT_PUBLIC_FORMSPREE_FORM_ID`)
- Static site generation with `generateStaticParams`

## Commands

```bash
npm install
npm run dev
npm run build
npm run seo:generate
npm run seo:verify
```

## Environment

Copy `.env.example` to `.env.local` and set:

- `NEXT_PUBLIC_FORMSPREE_FORM_ID`
- `NEXT_PUBLIC_SITE_URL=https://www.somaliaexpert.com`
- `GOOGLE_SITE_VERIFICATION` / `BING_SITE_VERIFICATION` (optional)

## SEO

Sitemap and robots.txt are generated by `scripts/generate-seo.ts` from `lib/seo/publicUrlInventory.ts`. Run `npm run seo:generate` after adding routes.
